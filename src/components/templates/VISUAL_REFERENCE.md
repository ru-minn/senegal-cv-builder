## Design Specifications

### Modern Template

**Layout:**
- Sidebar: 35% width
- Main content: 65% width
- Photo: 160px Ã— 160px (circular)

**Typography:**
- Font: system-ui, sans-serif
- Name: 4xl (36px) bold
- Section headers: 2xl (24px) bold
- Body: base (16px) regular

**Spacing:**
- Page padding: 2rem (32px)
- Section gaps: 2rem (32px)
- Item gaps: 1.5rem (24px)

**Colors:**
- Sidebar background: Accent color
- Sidebar text: White
- Main background: White
- Main text: Gray-900

### Classic Template

**Layout:**
- Full width content
- Photo: 128px Ã— 128px (square, optional)
- Two-column footer (skills/languages)

**Typography:**
- Font: Georgia, serif
- Name: 5xl (48px) bold
- Section headers: 2xl (24px) bold
- Body: base (16px) regular

**Spacing:**
- Page padding: 3rem (48px)
- Section gaps: 2rem (32px)
- Item gaps: 1.5rem (24px)

**Colors:**
- Background: White
- Headers: Accent color
- Body text: Gray-700
- Borders: Gray-300

## Print Specifications

Both templates use:
- Paper size: A4 (210mm Ã— 297mm)
- Print margins: 15mm all sides
- Font size (print): 12pt
- Color mode: CMYK (for professional printing)
- Resolution: 300 DPI equivalent

## Responsive Behavior

**Desktop (>1024px):**
- Full layout as shown
- Sidebar/columns preserved
- Optimal viewing experience

**Tablet (768px - 1024px):**
- Reduced sidebar width (Modern)
- Maintained layout structure
- Readable text sizes

**Mobile (<768px):**
- Single column layout
- Sidebar becomes header (Modern)
- Stacked sections
- Touch-friendly spacing

## File Size Estimates

**HTML/CSS (per template):**
- Modern: ~8KB
- Classic: ~7KB

**With Sample Data:**
- Modern: ~12KB
- Classic: ~11KB

**Exported PDF:**
- Typical: 50-150KB (depending on photo)
- With high-res photo: 200-500KB
- Without photo: 30-80KB
